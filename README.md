This is a basic Unreal Engine project, made for practicing purposes.
The project will contain a Dungeon Escape game, using prefabs and packages from Unreal.

In this project, we focused in the creation of 2 types of classes: Collectable Item (class that serves as a key), and Lock Actor (class where we put the key, in this case to move something).
Also, we worked with Shape Tracing (Raycast)
